DATA TYPES

  • Primitive
  • Non Primitive

PRIMITIVE

  • String (text) EX-let name = "pavithra";

  • Number (numbers) EX- let age = 21;

  • Boolean (true or false)EX- let isStudent = true;

  • Undefined (value not assigned) EX- let city;

  • Null (Empty value) EX- let data =null;

  • BigInt (large integers) EX- let num =1234567890n;

  • Symbol (Unique identifier) EX -let id =symbol("id");

NON-PRIMITIVE

  • Object EX- let person ={ name: "pavithra", age: 21 };
