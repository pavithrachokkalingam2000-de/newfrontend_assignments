WHAT IS HTML?


HTML stands for HyperText Markup Language. It is the standard language used to create and structure web pages.


MAIN PARTS OF HTML

  1. Root element of the page
  2. Contains title and other information
  3. Name shown in browser tab
  4. Visible content of the webpage
  5. Heading tag
  6. Paragraph tag

USES OF HTML

Creating websites

Designing web pages

Building forms

Displaying multimedia content

Structuring online applications

WHAT IS TAGS?

HTML tags are special keywords enclosed in angle brackets (< >) that tell the browser how to display content.

Define the structure of a webpage Tell the browser how to display content Usually come in pairs (opening + closing)


TAGS TYPES

Closing tag

Opening tag

Self closing tag

What is Block level and Inline level elements?

BLOCK LEVEL

Starts on a new line

Occupies full width

Can contain block and inline elements

INLINE LEVEL

Stays in the same line

Takes only required width

Mostly used inside block elements


What are the sementic tags and non sementic tags?

SEMENTIC TAGS

Semantic tags are meaningful tags.is describe your purpose and meaning of the content. the word those tags tell browser and developer what type of content present in the webpage.

Better readability → Easier for developers to understand code SEO (Search Engine Optimization) → Helps search engines understand your content Accessibility → Screen readers can interpret pages more clearly Cleaner structure → Organized and meaningful layout

NON SEMENTIC TAGS

Non-semantic tags are HTML tags that do not describe the meaning of the content inside them.

grouping elements

styling with CSS

adding JavaScript functions

whether the content is a header a footer navigation article sidebar
